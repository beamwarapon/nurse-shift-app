# Project Instructions

This project uses the [Agent Skills](skills/agent-skills/README.md) workflow library.

## Engineering Standards

- **Workflow:** Follow the [Using Agent Skills](skills/agent-skills/skills/using-agent-skills/SKILL.md) meta-skill for all development tasks.
- **Skill Discovery:** Refer to the [Skill Discovery](skills/agent-skills/skills/using-agent-skills/SKILL.md#skill-discovery) map to identify which skill applies to your current task.
- **Core Operating Behaviors:** Adhere to the [Core Operating Behaviors](skills/agent-skills/skills/using-agent-skills/SKILL.md#core-operating-behaviors) (Surface Assumptions, Manage Confusion, Push Back, Enforce Simplicity, Maintain Scope Discipline, Verify).

## Available Skills

The skills are located in `./skills/agent-skills/skills/`. Key skills include:
- [Spec-driven Development](skills/agent-skills/skills/spec-driven-development/SKILL.md)
- [Incremental Implementation](skills/agent-skills/skills/incremental-implementation/SKILL.md)
- [Frontend UI Engineering](skills/agent-skills/skills/frontend-ui-engineering/SKILL.md)
- [Test-driven Development](skills/agent-skills/skills/test-driven-development/SKILL.md)
- [Debugging and Error Recovery](skills/agent-skills/skills/debugging-and-error-recovery/SKILL.md)
